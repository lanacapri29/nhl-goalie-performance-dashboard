# NHL Goalie Performance Dashboard

## Project Overview
This project analyzes NHL goalie performance from the 2010-2015 seasons using a multi-page Power BI dashboard. The dashboard was designed to compare goalie performance across traditional metrics, workload indicators, and shot-volume trends.

## Tools Used
- SQL
- Power BI
- Power Query
- DAX
- Microsoft Excel

## Methods
- Cleaned and transformed five seasons of NHL goalie data
- Removed low-usage players and resolved inconsistencies across seasons
- Built DAX measures to support KPI tracking and performance comparisons
- Designed interactive slicers to filter and compare goalie performance
- Created visual pages evaluating save percentage, goals-against average, wins, workload, and shot volume

## Key Features
- Multi-page Power BI dashboard
- KPI pages for high-level performance summaries
- Interactive slicers for flexible analysis
- Visual comparisons of goalie workload and outcome metrics
- Data cleaning workflow using SQL and Power Query

## Business and Analytics Relevance
This project demonstrates data transformation, dashboard development, KPI design, and business intelligence reporting. The workflow is relevant to data analyst roles involving data preparation, reporting automation, performance monitoring, and stakeholder-facing insights.
